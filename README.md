# IBP Case Management System - Dagupan Chapter Prototype

**Research Project Title:**  
Design and Implementation of an Integrated Case Assignment, Tracking, and Document Management System for the Integrated Bar of the Philippines (IBP)

**Prototype Scope:** Dagupan Chapter only

**Common Case Types:**
- Civil (Annulment, Contract disputes, Property disputes, Damages)
- Criminal (Estafa, Theft/Robbery, Physical injury, Cybercrime)
- Labor (Illegal dismissal, Wage disputes)
- Family Law (Child support, Custody, VAWC)
- Land/Property (Land title, Boundary disputes, Ejectment)
