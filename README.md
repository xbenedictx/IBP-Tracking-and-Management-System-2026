# IBP Case Management System - Dagupan Chapter Prototype

**Project Title:**  
Design and Implementation of an Integrated Case Assignment, Tracking, and Document Management System for the Integrated Bar of the Philippines (IBP) - Dagupan Chapter

**Features Implemented:**
- Strict role-based secure login (Admin, Lawyer, Clerk, Viewer)
- Viewer can create new accounts
- Admin can manage users (placeholder)
- Automatic case assignment based on workload & expertise
- Real-time case tracking
- Secure document management
- Role-specific dashboards
- Cascading case type dropdowns (Civil, Criminal, Labor, Family Law, Land/Property)

**Sample Research Files Integrated:**
- Vol.65-Recent-Jurisprudence_2020-2021.pdf
- Case-Digest.pdf
